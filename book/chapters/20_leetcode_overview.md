# LeetCode Overview

This section contains **92 LeetCode problems** organized by topic, covering the LeetCode 75 study plan and additional practice problems.

## Topics Covered

| Category | Problems |
|----------|----------|
| **Array/String** | Two Sum, Product of Array, String Compression, etc. |
| **Two Pointers** | Container With Most Water, Move Zeroes, etc. |
| **Sliding Window** | Max Consecutive Ones, Maximum Vowels, etc. |
| **Stack** | Decode String, Asteroid Collision, Daily Temperatures |
| **Linked List** | Reverse Linked List, Odd Even, Delete Middle |
| **Trees** | Max Depth, LCA, Path Sum, BST operations |
| **Graphs** | Number of Islands, Provinces, Rotting Oranges |
| **Heap** | Kth Largest, Subsequence Score |
| **Binary Search** | Guess Number, Koko Eating Bananas, Peak Element |
| **DP** | House Robber, Unique Paths, Edit Distance, LCS |
| **Backtracking** | Letter Combinations, Combination Sum |
| **Bit Manipulation** | Single Number, Counting Bits, Minimum Flips |
| **Trie** | Implement Trie, Search Suggestions |
| **Intervals** | Non-overlapping Intervals, Burst Balloons |

## How to Use

1. **Study by pattern** - Problems are grouped by technique
2. **Track mistakes** - Note which patterns you miss
3. **Spaced repetition** - Re-solve after 1 day, then 1 week
